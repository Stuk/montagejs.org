An HTML5 framework for building modern Web Apps.

Quick Start Docs

Introducing Montage

The Montage framework lets you develop rich HTML5 applications optimized for today and tomorrow’s range of connected devices. You can build rich client-side UI’s, sharing packages and modules with your NodeJS server. Montage opens a new world of opportunity for web developers.

Some features

  • Components and HTML templates
  • 2-way data binding
  • Serialization
  • Event Management

Features

Reliable Components

MontageJS comes with a variety of self-contained, reusable, and reliable components for building native-quality HTML5 apps. You can use these components individually, nest them within others, or use our clean API to build your own. Learn more >

JSON Component Model

Montage models components as standalone web pages. The HTML of a component (the "template") contains a DOM skeleton and comes to life with the inclusion of a JSON formatted block or file—a component object model, distinct but connected to the DOM—that describes how to dress up the DOM with connections among components. Learn more >

Declarative Data Binding

MontageJS supports simple and two-way data binding between components, objects, and collections. Data binding helps keep your UI and model data in sync, so you have more time to spend on your actual data and application logic than on manually tying your data into the DOM. Learn more >

Managed Draw Cycle

MontageJS components participate in a managed draw cycle that separates reading and writing operations: DOM updates and queries are batched into separate code passes, thus limiting the number of DOM reflows while maximizing application performance. Learn more >

Event Delegation

MontageJS supports and extends the browser's native event handling mechanism using the concept of event delegation. This concept helps improve performance, provides simple event handling logic, and adds the ability to observe property changes. Learn more >
